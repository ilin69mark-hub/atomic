# Условная переменная в Go

Этот проект демонстрирует использование условной переменной (`sync.Cond`) в Go для координации между горутинами без использования атомарных операций.

## Описание задачи

Программа создает 1000 горутин, каждая из которых увеличивает общий счётчик на 1. После того, как все инкременты выполнены, программа выводит итоговое значение счётчика.

Вместо использования атомарных операций (`atomic.AddInt64`) используется условная переменная (`sync.Cond`) для синхронизации доступа к общим данным и уведомления о достижении целевого значения счётчика.

## Как запустить

Для запуска программы выполните следующую команду в терминале:

```bash
go run main.go
```

## Особенности реализации

- Используется `sync.Cond` для синхронизации между горутинами
- Реализована динамическая проверка достижения конечного значения счётчиком
- Каждая горутина захватывает мьютекс перед изменением разделяемых данных
- Используется `Broadcast()` для уведомления всех ожидающих горутин о достижении целевого значения
- В главной горутине используется цикл с проверкой условия для надёжного ожидания завершения
